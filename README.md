# liskit-dashboard
Simple lisk node dataviz

# Project at
http://txchain.github.io/liskit/

# Branches
    - master --> release branch
    - development --> development branch
    - feature-X --> new feature branch
    
# Run
    - clone
    - $ sudo npm install
    - $ bower install
    - $ grunt
    - launch a python simple server from the root
    - check on localhost:8000
